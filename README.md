# 公共建筑碳排放计算报告系统 V2.0

## 基于 GB/T 51366-2019 和 JS/T 303-2026

完整的建筑碳排放计算工具，包含 13 个工作表的 Excel 计算表格和 Word 报告模板，支持 Excel-Word 数据联动。

---

## 📁 文件结构

```
碳排放报告模板/
├── main.py                          # 统一入口脚本
├── generate_excel.py                # Excel 生成器（13个工作表）
├── generate_word.py                 # Word 报告生成器
├── create_vba_macros.py             # VBA 宏代码生成器
├── requirements.txt                 # Python 依赖
├── Excel_Word_Sync_Macros.bas       # VBA 宏代码文件
├── 公共建筑碳排放计算表.xlsx         # Excel 计算表格（输出）
├── 公共建筑碳排放计算报告.docx       # Word 报告模板（输出）
└── README.md                        # 本说明文档
```

---

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 生成所有文件

```bash
python main.py
```

或分别生成：

```bash
python main.py --excel   # 仅 Excel
python main.py --word    # 仅 Word
python main.py --vba     # 仅 VBA
python main.py --all     # 全部（默认）
```

## 🔗 Excel-Word 数据联动

### 设置步骤

1. **打开 Excel 文件** `公共建筑碳排放计算表.xlsx`
2. **导入 VBA 宏代码**
   - 按 `Alt+F11` 打开 VBA 编辑器
   - `文件` → `导入文件` → 选择 `Excel_Word_Sync_Macros.bas`
   - 保存为 `.xlsm` 格式（启用宏的工作簿）
3. **准备 Word 模板**
   - 确保 `公共建筑碳排放计算报告.docx` 与 Excel 在同一目录
4. **运行数据同步**
   - 按 `Alt+F8` → 选择 `UpdateWordReport` → 点击"运行"
   - 或使用工具栏上的"更新Word报告"按钮

### VBA 宏功能

| 宏名称 | 功能 |
|-------|------|
| `UpdateWordReport` | 将 Excel 数据同步到 Word 报告 |
| `ValidateData` | 检查必填字段是否填写完整 |
| `CreateToolbarButton` | 创建快捷工具栏 |

## 📝 填写指南

### 第一步：填写项目基本信息（工作表2）

| 字段 | 填写说明 | 示例 |
|-----|---------|------|
| 项目名称 | 完整项目名称 | XX市行政办公楼 |
| 建筑地址 | 详细到街道门牌号 | XX市XX区XX路XX号 |
| 建筑类型 | 从下拉选择或手动填写 | 办公建筑 |
| 建筑面积 | 地上+地下总面积（m²） | 25000 |
| 年用电量 | 从电费账单获取（kWh） | 1500000 |
| 年用气量 | 从燃气账单获取（m³） | 50000 |
| 年用热量 | 从热力账单获取（GJ） | 8000 |

### 第二步：填写能源消耗（工作表3-4）

1. 在**工作表3**中填写各类能源的年消耗量（黄色单元格）
2. 在**工作表4**中填写15+用能系统的详细参数
   - 暖通空调：冷负荷、热负荷、COP
   - 生活热水：用水人数、热水定额
   - 照明系统：各区域面积、功率密度
   - 电梯系统：载重量、速度、运行时间
   - 数据中心：IT设备功率、PUE值
   - 光伏系统：面板面积、辐射量、转换效率

### 第三步：填写建材数据（工作表5）

对50+建材类型填写消耗量（黄色单元格）：
- 结构材料：钢筋、混凝土、型钢
- 围护材料：砖、砌块、保温板、玻璃
- 装饰材料：石材、瓷砖、地板、涂料
- 安装材料：电缆、管材、风管
- 特殊材料：消防、安防、智能化设备
- 再生材料：再生混凝土、再生钢筋

### 第四步：填写建造拆除数据（工作表6）

对30+分部分项工程填写：
- 工程量
- 能耗系数（kWh/单位）

### 第五步：填写扩展模块（工作表7-8）

- **工作表7**：水资源消耗、废弃物处理
- **工作表8**：建筑碳汇、隐含碳排放

### 第六步：高级分析（工作表9-12）

- **工作表9**：情景分析（修改情景参数查看减排效果）
- **工作表10**：敏感性分析（识别关键影响因素）
- **工作表11**：对标与预测（与行业标准对比）
- **工作表12**：碳交易分析（碳配额、CCER开发）

### 第七步：查看汇总结果（工作表13）

查看所有自动计算结果：
- 各阶段碳排放量
- 单位面积碳排放
- 碳排放强度指标
- 核算结论

## 🔧 故障排除

### Q1: VBA 宏无法运行

**解决方案**：
1. 确认 Excel 已启用宏（文件 → 选项 → 信任中心 → 宏设置 → 启用所有宏）
2. 确认 VBA 代码已正确导入
3. 确认 Excel 文件已保存为 `.xlsm` 格式

### Q2: Word 报告数据未更新

**解决方案**：
1. 检查 Excel 中是否已填写完整数据
2. 运行 `ValidateData` 宏检查缺失字段
3. 确认 Word 模板与 Excel 在同一目录

### Q3: 计算公式显示错误值

**解决方案**：
1. 检查对应的黄色输入单元格是否为空
2. 检查输入的数据格式是否正确（数字、文本等）
3. 检查是否有除以零的情况（如建筑面积为0）
