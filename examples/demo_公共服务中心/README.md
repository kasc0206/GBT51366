## 文件说明

| 文件 | 说明 |
|-----|------|
| 计算表_公共服务中心.xlsx | 已填充数据的精细化计算表（39个工作表） |

## 使用步骤

1. 打开 `计算表_公共服务中心.xlsx` 查看示例数据
2. 根据需要修改黄色单元格中的数据
3. 使用同步工具将数据同步到Word报告

```bash
# 同步数据到Word
python ../main.py --sync
```
